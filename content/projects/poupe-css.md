Shared foundation for all CSS output in the Poupe stack, but
framework-agnostic — usable standalone in any TypeScript project.

Converts between camelCase and kebab-case property names, builds
and merges CSS declaration objects, formats nested rule blocks into
flat CSS strings, and provides type-safe helpers for common
shorthand expansions.
